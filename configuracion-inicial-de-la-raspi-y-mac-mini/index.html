Configuración inicial de la raspi y mac mini

En esta página mostraré el proceso de configuración de una Raspberry Pi y un Mac mini. Incluiré fotos y comentarios paso a paso para explicar cómo realizamos cada parte del procedimiento.

Raspberry Pi 400

Para la instalación de este Raspberry Pi hemos necesitado un cable USB-C para enchufarla a la corriente eléctrica, una tarjeta MicroSD que funciona como disco duro, un cable Micro-HDMI para ver la imagen en el monitor y un ratón para mover el cursor.

Le dijimos a la Raspberry Pi dónde estábamos. Seleccionamos España como país, Español europeo como idioma y la zona horaria de Madrid. Esto es fundamental para que el teclado escriba las teclas correctas (como la "ñ" o las tildes) y para que la hora del reloj sea la correcta.

Creamos la cuenta del "jefe" del ordenador. Antiguamente, las Raspberry Pi venían con un usuario por defecto llamado "pi". Por seguridad, ahora te obligan a crear uno nuevo. Nosotros decidimos usamos el nombre de usuario aulaateca26 y le pusimos la misma contraseña para no olvidarnos.

El sistema escaneó el aire buscando redes disponibles. Nosotros buscamos la nuestra en la lista y la seleccionamos para intentar conectarnos.

Una vez elegida la red, el sistema nos pidió la llave para entrar. Escribimos con cuidado la contraseña de la red del aula y pulsamos "Next" para que la Raspberry se conectara a internet.

Selección de navegador: El sistema nos da a elegir qué programa queremos usar para navegar por internet por defecto: Chromium (la base de Chrome) o Firefox.

Aviso de actualización: El asistente nos informa de que va a buscar actualizaciones para el sistema operativo y las aplicaciones. Debemos pulsar "Next" para confirmar.

Comprobando versiones: La Raspberry Pi se conecta a internet para comprobar si el software que tenemos instalado está desactualizado y compararlo con las versiones nuevas disponibles.

Descargando idiomas: Está descargando de internet los paquetes de traducción necesarios para poner el sistema en el idioma que seleccionamos (español).

Instalando idiomas: Una vez descargados los archivos de idioma, los está instalando y configurando para cambiar el texto de los menús.

Descargando el sistema: Este es el paso de “bajada”. Está descargando los archivos pesados con las mejoras de seguridad y las actualizaciones de los programas.

Instalando el sistema: Una vez descargados los archivos, está sustituyendo los viejos por los nuevos. Aquí es donde realmente se actualiza el sistema.

Proceso finalizado: El mensaje confirma que la instalación y actualización han terminado correctamente. Solo nos queda pulsar OK.

Fin de la configuración: El asistente nos confirma que la Raspberry Pi ya está lista.

Acción: El sistema pide pulsar "Restart" (Reiniciar). Esto es obligatorio para que todos los cambios que has hecho (idioma, hora, actualizaciones) se guarden y se apliquen correctamente.

Ya estamos dentro del sistema operativo (Raspberry Pi OS). Este es el fondo de pantalla por defecto.

Notificación: Arriba a la derecha. Ha salido un aviso gris que confirma que ya estamos conectados al Wi-Fi ("Wifi_AULA_ATECA") y se ha asignado una dirección IP.

Seguridad del navegador: Al abrir el navegador (Chromium) por primera vez, el sistema de seguridad (llamado "Keyring" o depósito de claves) te pide crear una contraseña maestra.

Para qué sirve: Es una "caja fuerte" digital donde el navegador guardará tus contraseñas de sitios web de forma segura. Si no quieres poner una, a veces basta con dejarlo en blanco y dar a continuar (aunque es menos seguro).

Probamos el rendimiento ejecutando un test de velocidad de internet.

Resultado: La prueba confirma que la antena Wi-Fi o el cable de red de tu Raspberry Pi 400 funcionan perfectamente, dando unos 113 Mbps de bajada y 133 Mbps de subida, lo cual es una conexión muy buena y rápida para este dispositivo.

Mac Mini

Para usar el Mac Mini hemos necesitado la propia unidad central conectada a la corriente con su cable de alimentación, además de un monitor unido mediante un cable HDMI o USB-C para poder ver la imagen. Por último, como el equipo no incluye periféricos integrados, ha sido imprescindible conectarle un teclado y un ratón externos para poder controlar el sistema.

Nada más encender el Mac, lo primero que hemos hecho ha sido seleccionar el Español como nuestro idioma principal y España como nuestra región. Al hacer esto, hemos configurado automáticamente el teclado, la moneda (Euros) y la hora local para que coincidan con nuestra ubicación.

En la pantalla de "Tratamiento", hemos elegido cómo queremos que el ordenador se dirija a nosotros (Femenino, Masculino o Neutro) para los mensajes de bienvenida y alertas del sistema.En la pantalla de "Tratamiento", hemos elegido cómo queremos que el ordenador se dirija a nosotros (Femenino, Masculino o Neutro) para los mensajes de bienvenida y alertas del sistema.

Hemos revisado el resumen de "Idiomas escritos y hablados" para confirmar que todo estaba correcto.

Nos ha aparecido una pantalla ofreciéndonos opciones de accesibilidad (Visual, Motriz, Auditiva). Como en este momento no necesitábamos ninguna adaptación específica para usar el equipo, hemos decidido pulsar "Ahora no" (o Continuar) para seguir adelante con la instalación estándar.

Para poder continuar, el Mac nos ha pedido conexión a internet. De la lista de redes disponibles, hemos buscado y seleccionado la nuestra. Esto es necesario para que el ordenador verifique la fecha, la hora y compruebe si hay actualizaciones críticas antes de seguir.

Nos ha aparecido una pantalla con el icono de dos personas dándose la mano titulada "Tus datos y la privacidad". Simplemente hemos pulsado "Continuar". Apple solo nos estaba informando de que usarán un icono especial cada vez que una aplicación intente acceder a nuestros datos personales.

El sistema nos ha preguntado si queríamos transferir información desde otro Mac antiguo o desde un PC con Windows. Como queríamos una instalación limpia (empezar de cero), hemos seleccionado la opción de "Ahora no" (botón inferior izquierdo).

  • Aquí el Mac nos pedía iniciar sesión con una cuenta de Apple (correo y contraseña). Como es un ordenador para un aula o uso general, no lo hemos vinculado a una cuenta personal.

    1. Hemos buscado y pulsado el enlace pequeño que dice "Configurar más tarde" (abajo a la izquierda).

    2. Cuando nos ha salido la ventana de advertencia preguntando "¿Seguro que no quieres iniciar sesión...?", hemos confirmado pulsando en "Omitir".

Hemos llegado a la pantalla de "Términos y condiciones". Para poder usar el equipo, hemos tenido que hacer clic en "Acepto" y confirmar nuestra decisión en la ventana emergente. Sin esto, Apple no nos deja avanzar.

Hemos configurado al usuario principal que va a controlar este Mac Mini.

    • Le hemos puesto de nombre completo y nombre de cuenta.

    • Hemos establecido una contraseña segura.

    • Hemos añadido una pista por si se nos olvida la contraseña.

Al darle a continuar, hemos terminado de crear el perfil de administrador y el Mac ha empezado a configurar el escritorio para que podamos empezar a trabajar.

El Mac nos ha preguntado si queríamos activar la localización. Esto sirve para que Mapas sepa dónde estamos y para que el ordenador ajuste la hora automáticamente.

Gracias a la conexión a internet (o a la localización), el mapa nos ha situado perfectamente en Pinto. Hemos confirmado que esa es nuestra ubicación para que el reloj del sistema marque la hora correcta de España peninsular.

Apple nos ha pedido permiso para recopilar datos de uso anónimos para mejorar sus productos.

Nos ha salido la pantalla informativa de "Tiempo de uso". Es una función que mide cuánto tiempo pasamos en cada aplicación. Como ahora solo queremos empezar a usar el ordenador, simplemente le hemos dado a "Continuar" .

Por fin hemos visto la pantalla de "Te damos la bienvenida al Mac".

Nada más llegar al escritorio, ha saltado una ventana automática. Como estamos usando un teclado externo, el Mac no estaba seguro de la distribución de las teclas. El asistente nos ha pedido pulsar la tecla situada justo al lado de la mayúscula izquierda para identificarlo correctamente.

Para terminar bien el trabajo, hemos ido a los Ajustes del Sistema > General > Actualización de software. Hemos visto que hay una actualización importante disponible (macOS Tahoe/Sequoia). Al igual que hicimos con la Raspberry Pi, lo ideal aquí es darle a "Actualizar ahora" para que el equipo esté seguro y tenga las últimas funciones.
