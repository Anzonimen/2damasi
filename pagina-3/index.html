Pomodoro

Proceso de resolución

 

En esta documentación explicaré los pasos y herramientas que utilicé para crear la aplicación Pomodoro.

En primer lugar, le pedí a ChatGPT que me ayudara a generar un prompt para Gemini.
Este fue el mensaje que le envié:

"Crea un prompt para Gemini en el que solicite el desarrollo de una aplicación de estudio Pomodoro en una sola página y en un único archivo HTML. Quiero que incluya funcionalidades prácticas, como un menú de apuntes, un menú de personalización y cualquier otra característica útil para una persona que desee estudiar. La aplicación debe ser web responsive y multiplataforma, compatible con teléfonos móviles."

Posteriormente, me preguntó si quería adaptar el prompt para que Gemini utilizara Tailwind CSS o Material Design inline, con el objetivo de lograr un aspecto visual más moderno. Le indiqué que, en el prompt, podía incluir cualquier tipo de recurso. Me actualizó el prompt a este

Posteriormente, accedí a Gemini para observar cómo desarrollaba la aplicación.

 

 

Al probar la aplicación, me di cuenta de que la opción para cambiar la fuente de los caracteres se seleccionaba correctamente, pero no aplicaba el cambio. Además, noté que faltaba una opción para cambiar el idioma a inglés, así que le pedí que actualizara la aplicación para incluir esa función.

 

 

Gemini actualizó la aplicación e implementó los cambios correctamente.

 

Posteriormente, le pedí a Gemini que añadiera la opción “Calculadora”, la cual debía ser completamente funcional. Además, le solicité que, en el apartado de ajustes, incluyera una opción para exportar el progreso en un archivo PDF, mostrando los datos en el idioma seleccionado en el que se encontrara la aplicación en ese momento.

 

 

Para añadir más funcionalidades, le pedí que creara una nueva opción llamada “Calendario”, en la cual, además de mostrar los días, aparecieran también las tareas pendientes de la opción “Tareas”.

Posteriormente, le pedí que implementara un horario semanal personalizable, de modo que el usuario pudiera añadir sus tareas y establecer la hora a la que desea realizarlas.

Luego, le pedí que creara una nueva opción llamada “Accesos Directos”, que permitiera al usuario acceder mediante iconos a herramientas útiles como WhatsApp Web, DeepL, Wikipedia, YouTube, y otras aplicaciones que pudieran resultar prácticas para un estudiante.
